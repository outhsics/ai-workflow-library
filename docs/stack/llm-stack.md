# 模型与聊天

## ChatGPT

- 适合做通用写作、结构化提纲、多轮整理
- 当我需要稳定语气和大范围梳理时优先使用

## Claude Code

- 适合长上下文代码修改、读仓库、做工程级改动
- 当任务需要直接落地成文件时更合适

## Codex

- 适合在终端工作流里直接查、改、跑、验
- 当目标是尽快把问题闭环时很高效

## Gemini / Perplexity

- 更偏搜索、资料补全、外部信息验证
- 适合补足“本地上下文之外”的内容

## Ollama

- 适合本地实验、隐私敏感内容或离线尝试
- 不一定是主力，但适合作为补充层

## 我会怎么选

1. 先判断是“信息问题”还是“执行问题”。
2. 信息问题优先用搜索和长上下文整理。
3. 执行问题优先进入编码代理和终端链路。
